# Auto Journal Project UI
![Vue.js](https://img.shields.io/badge/vuejs-%2335495e.svg?style=for-the-badge&logo=vuedotjs&logoColor=%234FC08D)
![Ionic](https://img.shields.io/badge/Ionic-%233880FF.svg?style=for-the-badge&logo=Ionic&logoColor=white)

This is the front-end for the AutoJournal project.
Uses Vue3 and Ionic Framework. Currently runs on web, but mobile iOS and Android builds are planned for the future, using Capacitor framework.

The server is a separate repository (https://github.com/Bujdi22/CarRecords)

## Environment variables
Save this in root directory in a file named .env
```env
VITE_API_URL=http://localhost:8080
```


## How to run

```bash
# Using node 22

# install ionic cli if not already
# npm install -g @ionic/cli

npm install

ionic serve

 
```
